#easycms  持续开发中。。。

### 简介
Java EasyCms 使用最简单性能最高的框架，将cms系统简单到极致，灵活的栏目扩展，快速的构建普通网站
###用到的框架工具
java，servlet，guice4， mybatis3， jstl，bootstrap2，bootstrap3，jerichotab，jqGrid，jquery，jquery-validation，layer，My97DatePicker，ueditor，zTree等等
